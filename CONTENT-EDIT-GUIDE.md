# 作品集文案编辑清单

这份清单只整理网页里会展示给浏览者看的文字。实际替换时主要改两个文件：

- 首页、能力页、入口页与按钮：`index.html`
- 四条路线的每页标题和正文：`app.js`

不要修改英文 class 名、`data-route`、`aria-label`、`id` 与脚本变量；它们负责交互。

## 最实用的编辑方式

1. 先在 `app.js` 里只改 `title`、`sub`、`body`。
2. 再改 `index.html` 里的首页身份信息和能力标签。
3. 每改完一组，打开 `http://127.0.0.1:5173/` 刷新查看。
